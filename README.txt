Progevity PA — Restore Pack v1
------------------------------
This pack restores:
• Light theme by default
• Navbar (Home / Our Services / About / Book / Patient Portal)
• Home hero video (hero-home.mp4 with hero-poster.jpg fallback)
• Six feature cards, Testimonial strip, FAQ
• Booking page wired to your SimplePractice links

Drop-in instructions:
1) Stop the dev server.
2) Copy files into your project, replacing same paths.
3) Ensure /public has hero-home.mp4 and (optional) hero-poster.jpg.
4) Start dev server: npm run dev
